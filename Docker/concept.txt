How to share images :

- With internet
  
  - Step 1: Tag image
    docker tag myimage:v1 dinesh/myimage:v1
    
  - Step 2: Login to Docker Hub
    docker login
    
  - Step 3: Push image
    docker push dinesh/myimage:v1
    
  - Step 4: Pull image on another machine (optional)
    docker pull dinesh/myimage:v1
    
- Without internet
  
 - Step 1: Save image
   docker save -o ./my_image_backup.tar my_image:v1
   
 - Step 2: Transfer file to another machine
   scp ./my_image_backup.tar user@target_machine:/path/to/destination
   
 - Step 3: Load image
   docker load -i ./my_image_backup.tar
   
 - Step 4: Verify
   docker images
